# Flutter POS - Hello World

Project ini berisi aplikasi Flutter **Hello World** yang siap dijalankan.

## Struktur utama

- `lib/main.dart`: Entry point aplikasi dan tampilan utama Hello World.
- `test/widget_test.dart`: Widget test sederhana untuk memverifikasi UI.
- `pubspec.yaml`: Konfigurasi dependencies Flutter.

## Cara menjalankan

1. Install Flutter SDK (stable) dan pastikan `flutter` ada di PATH.
2. Jalankan perintah berikut:

```bash
flutter pub get
flutter run
```

## Cara build

Contoh build APK release:

```bash
flutter build apk --release
```

Contoh build web:

```bash
flutter build web
```
